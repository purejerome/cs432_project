All-Errors Suite (Reachable)
----------------------------
This suite triggers every user-facing ErrorList_printf message present in p3-analysis.c,
except the two "Internal error: unhandled ... operator" messages which are not reachable
through valid Decaf syntax.

Mapping (filename -> message kind):
01  err01_symbol_undefined_var.decaf          -> Symbol 'y' undefined on line N
02  err02_symbol_undefined_func.decaf         -> Symbol 'undef' undefined on line N
03  err03_duplicate_symbols_global.decaf      -> Duplicate symbols named 'a' ...
04  err04_duplicate_params.decaf              -> Duplicate symbols named 'a' ... (param scope)
05  err05_no_main_function.decaf              -> Program does not contain a 'main' function
06  err06_main_not_function.decaf             -> Symbol 'main' is not a function ...
07  err07_main_must_return_int.decaf          -> 'main' must return an integer
08  err08_main_takes_params.decaf             -> 'main' must take no parameters
09  err09_if_condition_not_bool.decaf         -> Type mismatch: bool expected but int found ...
10  err10_while_condition_not_bool.decaf      -> Type mismatch: bool expected but int found ...
11  err11_assignment_incompatible.decaf       -> Type mismatch: int is incompatible with bool ...
12  err12_void_variable.decaf                 -> Void variable 'v' ...
13  err13_array_length_nonpositive.decaf      -> Array 'a' ... must have positive non-zero length
14  err14_local_array_disallowed.decaf        -> Local variable 'a' ... cannot be an array
15  err15_void_func_returning_value.decaf     -> Invalid non-void return from void function ...
16  err16_int_func_void_return.decaf          -> Invalid void return from non-void function ...
17  err17_return_type_mismatch.decaf          -> Type mismatch: int expected but bool found ...
18  err18_call_non_function.decaf             -> Invalid call to non-function 'foo' ...
19  err19_invalid_arg_count.decaf             -> Invalid number of function arguments ...
20  err20_param_type_mismatch.decaf           -> Type mismatch in parameter k of call to 'f' ...
21  err21_nonarray_as_array.decaf             -> Non-array 'x' accessed as an array ...
22  err22_array_without_index.decaf           -> Array 'a' accessed without index ...
23  err23_function_as_variable.decaf          -> Function 'f' accessed as a variable ...
24  err24_index_type_mismatch.decaf           -> Type mismatch: int expected but bool found ...
25  err25_unary_neg_nonint.decaf              -> Type mismatch: int expected but bool found ...
26  err26_unary_not_nonbool.decaf             -> Type mismatch: bool expected but int found ...
27  err27_binary_int_expected.decaf           -> Type mismatch: int expected but bool found ... (right)
28  err28_binary_bool_expected.decaf          -> Type mismatch: bool expected but int found ... (both sides)
29  err29_equality_incompatible.decaf         -> Type mismatch: int is incompatible with bool ...
30  err30_break_outside_loop.decaf            -> Invalid 'break' outside loop ...
31  err31_continue_outside_loop.decaf         -> Invalid 'continue' outside loop ...
